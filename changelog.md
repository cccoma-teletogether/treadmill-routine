[TASK-001] 2026-05-23 코드 분석 및 버그 수정
- 속도 슬라이더 조작 시 설정이 즉시 저장되어 새로고침해도 유지됩니다.
- 호출되지 않던 데드 코드(`saveSpeedSettings()`)가 제거되었습니다.
- 루틴 관리에서 루틴 선택 시 불필요한 알림 없이 바로 타이머 탭으로 이동합니다.
- 현재 설정 표시에 배율이 적용된 속도(km/h)와 퍼센트가 함께 표시되며, 배율 1.0은 생략됩니다.
- 루틴 복제 시 수정된 루틴이 있으면 수정본 기준으로 복제됩니다.

[TASK-002] 2026-05-23 타이머 버튼 UX 개선
- 버튼이 2개로 통일되었습니다: 토글 버튼 + 초기화 버튼.
- 토글 버튼은 시작 → 일시정지 → 계속 순으로 상태에 따라 바뀝니다.
- 초기화 버튼을 누르면 타이머가 완전히 정지되고 처음 상태로 돌아갑니다.
- 운동 완료 시 토글 버튼이 자동으로 "시작"으로 복귀합니다.

[TASK-003] 2026-05-23 "피트니스" 시간 표시 스타일 추가
- 설정 관리의 시간 표시 스타일에 디지털/아날로그 외 "피트니스" 옵션이 추가되었습니다.
- 파란색 파이 차트 다이얼이 경과 시간을 부채꼴(0→360°)로 표시합니다.
- 단계 남은 시간과 전체 남은 시간이 함께 표시됩니다.
- 선택한 스타일은 localStorage에 저장되어 재방문 시에도 유지됩니다.
- 피트니스 스타일이 설정에서 선택되지 않던 버그를 수정했습니다.
